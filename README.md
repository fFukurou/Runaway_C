# Runaway_C
Rendition of the classic Pac-Man game.

-Move with w, a, s ,d  

-Pick up the pill (power-up) to be able to explode the tiles adjascent to you  

-If the ghosts touch you, you die  

-Try to blow up both of them. You only get one shot
